# Answering Multiple Choice Questions

## Description
This procedure explains how to effectively answer multiple choice questions, using a step-by-step approach demonstrated with a real example.

## Prerequisites
* Know how to find the main idea
* Know how to find meaning of words in context  
* Know how to mark evidence in text

## Steps

### 1. Read the Question
1. Focus on the question stem
2. Identify keywords and important details
3. Understand what is being asked before looking at choices

### 2. Review Answer Choices
1. Read all options carefully
2. Identify differences between similar answers
3. Mark obviously incorrect answers for elimination

### 3. Analyze Evidence
1. Recall relevant information from source material
2. Look for specific supporting details
3. Compare evidence to remaining answer choices

### 4. Select Answer
1. Choose most accurate and complete option
2. Verify selection with evidence
3. Mark answer clearly using required method

## Example

### Sample Question
Which event served as the primary catalyst for Japan's rapid modernization in the 19th century?

* A) The desire to preserve samurai traditions and ancient customs
* B) The arrival of Commodore Perry's "Black Ships" and the threat of Western military power
* C) The natural evolution of Japanese society and culture
* D) The personal ambitions of Emperor Meiji

### Example Analysis

#### 1. Question Analysis
* Key phrase: "primary catalyst"
* Time period: 19th century
* Looking for: Initial cause of modernization

#### 2. Evidence Review
* Text mentions Perry's arrival as "pivotal moment"
* Japanese had never seen steam-powered vessels
* Perry threatened military action
* Changes followed this event directly

#### 3. Choice Evaluation
* A) Contradicts text - samurai system was abolished
* B) Matches text's sequence of events
* C) Conflicts with text's description of external pressure
* D) Text shows emperor as symbol, not initiator

#### 4. Answer Selection
* Correct answer: B
* Supporting evidence: Text directly connects Perry's arrival to subsequent modernization
* Clear cause-and-effect relationship established

## Related Information
* This method can be applied to any multiple choice question
* Practice improves speed and accuracy
* Time management remains important

## Notes
* Always read instructions carefully
* Mark answers according to test requirements
* Review answers if time permits
