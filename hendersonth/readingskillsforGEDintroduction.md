# Reading Skills for the GED Test: A Comprehensive Guide

## Description
This guide provides a structured approach to developing essential reading comprehension skills needed for success on the GED test. It consists of multiple procedures that build upon each other to help you master reading comprehension and effectively answer test questions.

## Overview
Before attempting to answer GED reading comprehension questions, you must first master three fundamental skills. Each skill has its own detailed procedure guide:

### Essential Skill Procedures
1. Identifying Main Ideas
  * Develop skills to find central themes
  * Learn to distinguish main ideas from supporting details
  * Practice summarizing key concepts

2. Marking Text Evidence
  * Learn effective annotation techniques
  * Master methods for highlighting key information
  * Practice identifying relevant evidence

3. Finding Word Meaning in Context
  * Learn to understand unfamiliar words using context clues
  * Master techniques for analyzing word usage
  * Practice identifying word meanings without a dictionary

### Final Skill Application
After mastering the above skills, proceed to:

4. Answering Multiple Choice Questions
  * Apply all previous skills
  * Learn systematic approach to analyzing questions
  * Practice with sample passages and questions

## Prerequisites
* Basic reading ability
* Understanding of English grammar
* Dedication to completing all procedure guides in order

## Important Notes
* Complete guides in sequential order
* Master each skill before moving to the next
* Practice regularly with provided examples
* Return to earlier guides if needed for review
