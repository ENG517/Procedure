# Hi set skills

Below are skills needed for the Social Studies Hi-set

<ul>
<li> Understanding the authors point of view on reading passages </li>
<li> Explaining the difference between facts, opinions and values </li>
<li> Understanding the princples of geography (more important that memory of geographical facts) <li>
<li> Understanding of diagrams, statistics and persuasive images </li>
<li> Understanding of personal finance, economics and the ability to comprehend documents including taxation forms and bank statements </li>
</ul>
