# Brayden Marsh - User Scenario - Logic Pro

**Scenario**: 

### Task 1

#### How to create a vocal track.

John is new to logic and wants to create a vocal track. He is completely novice to the interface and needs instructions on how to create a vocal track.

### Task 2

#### Locating vocal presets within Logic.

John knows how to create a vocal track but is novice to vocal production. He doesn’t know where to start when it comes to making a vocal chain. He thinks it is easiest to use Logic’s vocal presets but doesn’t know where to find them. He needs instructions on where to look for vocal presets and how to try each of them.

### Task 3

#### Understand how to add effects to a vocal chain. 

John has some experience with vocal production but he wants to be more creative and unique with his voice. He has been using vocal presets but wants more control over his vocal chain. He needs instructions on where to add effects so that he can experiment with different production styles.
