# Howrse: Breeder's How-To Manual

For this project, I use free-to-play browser game Howrse as the point of reference for writing my procedures. Howrse is an online horse simulation game where you can breed and sell horses, manage an equestrian center, collect in-game items, compete on teams, and participate in weekly promotional contests to earn in-game prizes.

My procedures each focus on a different task that players can perform in the game:
1. Reserving and applying an affix to a horse ([Preview](https://github.com/ENG517/Procedure/blob/main/loyc/p-howrse-reserve-apply-affix.md))
2. Covering a mare with a public covering offer ([Preview](https://github.com/ENG517/Procedure/blob/main/loyc/p-howrse-cover-mare.md))
3. Maximizing the Best Linear Unbiased Prediction (BLUP) value of a horse ([Preview](https://github.com/ENG517/Procedure/blob/main/loyc/p-howrse-maximize-blup.md))
