## How do your procedures follow the WTGA Staging, Coaching, and Alerting stylistic conventions? Provide some specific examples that connect back to the criteria from Hart-Davidson.

Staging: I began each procedure with a starting point, effectively preparing the user by giving them the prerequisites to complete the action. For example, the need to import libraries or define the data. This staging, especially the one with Matplotlib ensures the users understand what they need to complete the task.

Coaching: Throughout the steps, I provided explanations and screenshots to coach the user through the procedures. Alerting: I used alerting by highlighting critical commands that are essential to success, such as reminding the user to use "plt.show". Without "plt.show" the plot does not appear.
