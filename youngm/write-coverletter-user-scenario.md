# User Scenario - Writing A Cover Letter

**Scenario**: 
User's can range from someone who has never written a cover letter before to someone who has written a cover letter before but needs a reminder of the contents.

User's are likely looking for instructions and examples, however because the content and context of a cover letter changes depending on the writer and on the position the letter is written for, the guide is not as comprehensive as detailed step by step guide. 

Potential Scenario: A high schooler is looking to apply to their very first job. Everything avaliable to them are entry level positions. When applying for these jobs, the user discovers that they need to submit a cover letter with the application. Having never written a cover letter before, the user encounters the guide and gains the basic understanding of the purpose of a cover letter. They also gain a basic structure for the letter and clarification on the ideal length. The user finishes composing a basic cover letter, and sends it in alongside their application.
