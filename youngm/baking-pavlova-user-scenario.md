# User Scenario - Making A Pavlova

**Scenario**: An individual, who is a potential user, is looking for instructions on how to create a pavlova. 

It is unlikely that a brand new user would use this guide because a pavlova is not a common dessert. Additionally, a brand new user may be intimidated by the long amount of time it takes to complete. 

This is a hobbyist guide, so the user likely has an interest in baking.The user likely has access to certain equipment such as some version of a kitchen mixer and an oven. 

Possible Scenario: An individual is hosting a dinner party, one of the guests has a gluten allergy. The user finds the guide while searching for gluten-free dessert options. Because they have an interest in cooking and baking, they feel capable of following the recipe. The user follows the guide, gathering the ingredients, assembling the batter, and baking the pavlova in the oven. The recipe is a success and the final product is served at the user's event.
